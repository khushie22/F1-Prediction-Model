About F1 Singapore GP Predictor

This application uses machine learning to predict the winner of the 2025 Singapore Grand Prix based on comprehensive historical data analysis.

Data Sources

  • Historical Race Results (2008-2024): Singapore GP results from FastF1 and Ergast API
  • Driver Performance: Championship standings, consistency metrics, and track-specific performance
  • Team Performance: Constructor standings, reliability data, and car performance trends
  • Weather Data: Temperature, humidity, precipitation, and weather conditions
  • Track Characteristics: Singapore circuit specifications and historical race patterns

Machine Learning Approach

Our prediction model has tested multiple algorithms and selected the best performing one which is:

  • XGBoost: Gradient boosting for complex pattern recognition

Features Used

Driver Features
  • Singapore GP history and success rate
  • Recent form (last 5 races)
  • Career statistics and consistency
  • Experience and age factors
Team Features
  • Singapore GP team performance
  • Recent team form and reliability
  • Car performance characteristics
  • Team championship standings
Track Features
  • Singapore circuit characteristics
  • Overtaking difficulty
  • Tire degradation patterns
  • Safety car probability
Weather Features
  • Temperature and humidity effects
  • Rain probability and impact
  • Weather-related performance patterns
  • Historical weather impact

Model Performance

The model is evaluated using multiple metrics:

  • Accuracy: Overall prediction correctness
  • Precision: True positive rate for winner predictions
  • Recall: Sensitivity to actual winners
  • F1-Score: Harmonic mean of precision and recall
  • AUC: Area under the ROC curve

Limitations

  • Predictions are based on historical data and may not account for unexpected events
  • Driver transfers and team changes between seasons may affect accuracy
  • Weather conditions are estimated based on historical averages
  • Technical issues, crashes, or other race incidents are not predictable

Technical Details

Built with: Python, Flask, scikit-learn, pandas, FastF1, Bootstrap

Data Collection: Automated via APIs with caching for performance

Model Training: Cross-validation and hyperparameter optimization

Deployment: Local Flask application with responsive web interface

Disclaimer

This is a machine learning experiment for educational purposes. Predictions should not be used for betting or gambling. F1 is unpredictable, and many factors can influence race outcomes that are not captured in historical data.
